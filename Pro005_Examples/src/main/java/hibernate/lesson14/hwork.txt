Задание.

Спроектировать и реализовать приложение по управлению магазином ноутбуков.

1. В приложении должны быть следующие сущности:

- CPU со следующими полями:
    - Серийный номер
    - Марка процессора
    - Частота процессора
- Memory со следующими полями:
    - Серийный номер
    - Тип оперативной памяти
    - Объем оперативной памяти
    - Частота оперативной памяти
- Vendor со следующими полями:
    - Название производителя
    - Описание производителя

- Notebook (основная сущность):
    - Серийный номер модели
    - ЦПУ
    - Оперативная память
    - Производитель
    - Стоимость за единицу
    - Дата производства

- Store (основная сущность)
    - Номер партии
    - Дата поступления
    - Ноутбук
    - Количество на складе

2. Реализовать DAO для сущностей CPU, Memory, Vendor и Notebook,
   учитывать CRUD-методы и получение всех записей.

3. Реализовать Service для сущности Notebook, со следующим
   функционалом:
   -  Поиск по производителю ноутбука
   -  Поиск по дате производства
   -* Поиск по характеристикам ноутбука
        (Например по обьему оперативной памяти или
         по частоте процессора и т.д.)
   Реализовать Service для Store:
   -  Создание новой партии ноутбуков
   -  Продажа ноутбука со склада (желательно добавить
      проверку наличия ноутбука)
   -  Поиск ноутбука по номеру партии
   -  Поиск ноутбука по дате поступления

Приветствуется создание приложения в JavaFX. Если кто захочет
сможет продемонстрировать его на проекторе в классе.
